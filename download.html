Download

libdill

http://libdill.org/libdill-1.2.tar.gz

$ wget http://libdill.org/libdill-1.2.tar.gz
$ tar -xzf libdill-1.2.tar.gz 
$ cd libdill-1.2
$ ./configure
$ make
$ sudo make install
