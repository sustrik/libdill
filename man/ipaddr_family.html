NAME

ipaddr_family - returns family of the IP address

SYNOPSIS

int ipaddr_family(const struct ipaddr *addr);

DESCRIPTION

Returns family of the address, i.e. either AF_INET (IPv4) or AF_INET6 (IPv6).

RETURN VALUE

Family of addr.

ERRORS

No errors.

EXAMPLE

int family = ipaddr_family(&addr);
